# 桌面版安装与首次打开

GitHub Actions 会生成三个独立安装包：

| 文件 | 适用设备 |
|---|---|
| `SearchIndexSubmitter-macOS-Intel.zip` | Intel 芯片 Mac |
| `SearchIndexSubmitter-macOS-Apple-Silicon.zip` | M1、M2、M3、M4、M5 等 Apple Silicon Mac |
| `SearchIndexSubmitter-Windows-x64.zip` | 64 位 Windows 10/11 |

## 如何确认 Mac 芯片

点击苹果菜单 →“关于本机”：

- 显示“芯片 Apple M…”：下载 Apple Silicon 版；
- 显示“处理器 Intel…”：下载 Intel 版。

## macOS 首次打开

1. 解压 ZIP。
2. 把 `SearchIndexSubmitter.app` 拖入“应用程序”。
3. 第一次打开如果 macOS 提示无法验证开发者，请关闭提示。
4. 打开“系统设置”→“隐私与安全性”。
5. 在安全性区域确认打开该应用；也可以在 Finder 中右键应用并选择“打开”。

当前自动构建使用临时签名，没有 Apple Developer ID 公证。正式商业发行前应配置开发者证书、Hardened Runtime 和 Apple Notarization。

## Windows 首次打开

1. 解压 ZIP。
2. 双击 `SearchIndexSubmitter.exe`。
3. 如果 Windows SmartScreen 提示未知发布者，请确认文件来自本项目官方 GitHub Actions 后再选择运行。

当前自动构建没有商业代码签名证书。正式发行前应购买代码签名证书并为 EXE 签名。

## 凭据和数据

桌面包不会预置任何平台凭据。每位用户需在软件“平台凭据”中配置自己的账号信息。

完整步骤参见：[用户配置与凭据获取指南](搜索引擎一键提交工具_用户配置与使用指南.md)。
