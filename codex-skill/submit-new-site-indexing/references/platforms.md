# Platform Rules

| Provider | Duplicate check | Submission |
|---|---|---|
| Google | URL Inspection API | Search Console Sitemap API |
| Bing | Webmaster GetUrlInfo | Webmaster URL API or IndexNow |
| Yandex | Unknown unless a supported inspection API is configured | Webmaster Sitemap API or IndexNow |
| Baidu | No stable public exact-URL inspection API | Ordinary indexing URL API |
| IndexNow participants | No shared indexing-status query | IndexNow batch URL API |
| 360 / Shenma | No stable public exact-URL inspection API | Manual webmaster portal |

Treat unknown as eligible for submission. Notification never guarantees crawling, indexing, or ranking.
